# Atividades- Aula 03- Regivaldo Araújo

Atividade resolvida em Julia a partir dos notebooks `S03-01.ipynb` a `S03-06.ipynb`,
com implementacoes dos metodos numericos pedidos nos capitulos 7, 8, 9, 10, 11 e 13.

Referencia do material:
https://transp-or.epfl.ch/books/optimization/html/index.html

## Como executar

```powershell
julia atividade_aula3.jl
```

O script usa apenas bibliotecas padrao do Julia (`LinearAlgebra` e `Printf`).

## Entregaveis gerados

- `atividade_aula3.jl`: codigo completo da atividade.
- `resultados/RESULTADOS.md`: respostas e tabelas das iteracoes.
- `resultados/newton_x2menos2.svg`: grafico das iteracoes do metodo de Newton para `F(x)=x^2-2`.
